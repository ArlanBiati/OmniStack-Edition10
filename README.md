<h3>Feito com as tecnologias:</h3>

<p>
Backend - <a href="https://nodejs.org/en/">Node.js</a></br></br>
Frontend - <a href="https://pt-br.reactjs.org/">ReactJS</a></br></br>
Mobile - <a href="https://facebook.github.io/react-native//">React Native</a>

O projeto tem como objetivo criar um radar para que empresas ou juridicas possam encontrar estes profissionais através de suas tecnologias e distancia em Km.

=> Utilizamos o método real time para o App;</br>
=> Utilizamos método de geolocalização para cadastrar a posição de cada dev;
